## 1. 产品概述

「CardTalk · 字卡聊天室」是一款单页 Web 聊天应用，玩家扮演"我"与一位虚拟角色对话。核心设定是：**对方只能通过有限的「字卡」来回应你**——她不会打字，只会从随身携带的卡牌堆里挑一张递给你。

为了让对话更有沉浸感，玩家可以随时切换到**对方视角**查看整段聊天记录（看到她眼中的你），并像拿起对方手机一样查阅她的"小手机"——里面有她的身体状况、心情、工作、出行四个生活模块，揭示她的真实状态。

- **目标用户**：喜欢沉浸式叙事、慢节奏文字交流、轻量角色扮演的玩家。
- **核心价值**：用"字卡限流"制造沟通张力，用"偷看手机"提供信息差与情感暗线，让简单聊天变成一场需要观察、推理与共情的体验。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 说明 |
|------|------|
| 我（玩家） | 可自由输入消息，可切换视角、可拿起对方手机查看 |
| 她（虚拟角色） | 仅能用预设字卡回复，拥有独立的手机数据 |

### 2.2 功能模块
1. **聊天室**：双向消息流，对方以"字卡"气泡呈现；我的输入框在底部。
2. **视角切换**：顶部切换按钮，在"我的视角 / 她的视角"间切换，切换后所有消息左右排布、头像、气泡配色互换。
3. **小手机抽屉**：右侧可拉出/收起的迷你手机，包含四个生活模块。
4. **字卡库**：对方按情绪与话题动态挑选卡片回复，玩家可见可选卡片暗影。

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 聊天室 | 消息流 | 我发送文字气泡；她发送字卡气泡（卡片样式：边框、印戳、卡片名） |
| 聊天室 | 输入区 | 我的输入框 + 发送按钮；她的"翻卡"提示区（不可输入，仅提示她正在挑卡） |
| 聊天室 | 视角切换 | 顶部 Toggle：「我 ↔ 她」，切换后整页重排并播放过渡动画 |
| 小手机 | 身体状况 | 体温、心率、睡眠、疲惫度，带趋势小图 |
| 小手机 | 心情 | 当前心情标签、心情曲线、今日关键词 |
| 小手机 | 工作 | 今日工作项、完成进度、加班状态 |
| 小手机 | 出行 | 当前位置、今日行程、天气 |

## 3. 核心流程

玩家进入聊天室 → 输入消息 → 她"翻卡"后送出字卡 → 玩家可随时切换视角查看对方眼中的对话 → 也可拉出小手机查看她的真实状态（状态会随对话进展微调）。

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：暖色调日系手账 + 轻复古印刷质感。米黄纸张底色 + 墨黑文字 + 一抹朱砂红印章色作为强调。
- **配色**：主色 `#F4ECD8`（米黄纸），文字 `#2B2B2B`（墨黑），强调 `#C73E3A`（朱砂红），辅助 `#6B8E7F`（竹绿）。
- **字体**：标题用 `Noto Serif SC`（衬线，书卷气），正文用 `Noto Sans SC`，卡片名用 `ZCOOL XiaoWei`（手写印章感）。
- **卡片样式**：字卡呈实体卡牌形态——圆角矩形、细描边、左上角小印章、右下角卡片编号；悬浮时微抬升 + 阴影。
- **布局**：桌面优先，左主聊天区 + 右侧可拉出小手机抽屉；移动端自适应为单列 + 底部抽屉。

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 聊天室 | 消息流 | 纸质纹理背景，气泡有针孔/胶带装饰，字卡带印章 |
| 聊天室 | 视角切换 | 顶部胶囊 Toggle，切换时整页 3D 翻转动画 |
| 小手机 | 手机壳 | 类 iPhone 外壳，刘海/状态栏，内部分屏 |
| 小手机 | 身体状况 | 体温计插画 + 心率折线 mini 图 |
| 小手机 | 心情 | 心情 emoji 标签 + 24h 心情曲线 |
| 小手机 | 工作 | 任务清单 + 进度条 |
| 小手机 | 出行 | 地图风格卡片 + 天气图标 |

### 4.3 响应式
- 桌面（≥1024px）：左聊天 + 右手机并排，手机常驻侧栏可滑出。
- 平板（768–1024px）：手机改为底部抽屉。
- 移动（<768px）：单列，聊天为主，手机为全屏覆盖抽屉。
